# PeriodicSync プラグイン for Movable Type

サーバー配信を1時間単位で定期実行可能にします。

![スクリーンショット](https://github.com/masiuchi/mt-plugin-periodic-sync/raw/master/ScreenShot.png)

## 動作確認環境

* MT6 クラウド

※ 多少スタイル崩れがありますが、MT7 でも動作しそうです。
